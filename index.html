클린쿨타임을 만드는 사람

클린쿨타임은 1인 개발자가 만든 생활 편의 앱입니다. 화려한 기능보다 “꼭 필요한 일을, 사용자 입장에서, 정직하게” 해내는 것을 더 중요하게 생각하며 만들었습니다.

개발하며 지키는 원칙

1

사용자 데이터 존중

꼭 필요한 정보만 다루고, 가능하면 사용자의 기기 안에만 머무르도록 설계합니다. ‘클린쿨타임’은 이용자가 입력한 정보를 외부 서버에 저장하지 않고 오직 사용자 기기 안에만 보관합니다.

2

정직한 안내

앱이 실제로 하는 일과 하지 않는 일을 부풀리거나 숨기지 않고 그대로 설명합니다. 수익 모델(예: 제휴 링크)이 있다면 명확히 고지합니다.

3

부담 없는 사용

회원가입 없이 무료로 쓸 수 있는 가벼운 도구를 지향합니다.

앱 소개

클린쿨타임

Clean Cooltime · 가족용 교체주기 알리미

“또 갈아야 하는 걸 깜빡했네”라는 일상의 작은 후회를 줄여주는 가정용 유틸리티 앱입니다. 칫솔, 수건, 정수기 필터처럼 일정한 ‘교체 주기’가 있는 생활용품은 막상 때가 되면 기억하기 어렵습니다. 클린쿨타임은 이런 물건들을 한 번 등록해 두면, 교체 시기가 다가올 때 기기 알림(로컬 알림)으로 미리 알려줍니다.

가장 큰 신뢰 포인트. 이용자가 입력한 정보는 기기를 떠나지 않습니다. 가족 별칭, 항목명, 카테고리, 교체 주기, 마지막 교체일, 교체 이력, 가격(선택), 여분 재고 등 입력한 모든 정보는 오직 사용자 기기 내부의 로컬 저장소(IndexedDB)에만 저장됩니다. 이용자 데이터를 저장·수집하는 자체 서버가 없고, 회원가입·로그인도 없으며, 개발자나 제3자에게 전송·수집되는 입력 정보가 없습니다. 현재 분석 도구·추적·광고 SDK를 사용하지 않습니다.

참고: 앱에 쓰이는 글꼴과 라이브러리는 모두 앱 안에 포함(번들)되어 있어, 앱이 자동으로 외부 서버나 CDN에 접속하지 않습니다. 외부 통신은 이용자가 ‘장보기’에서 쿠팡 링크를 직접 누를 때에만 일어납니다.

데이터는 사용자가 직접 관리할 수 있습니다. 백업(JSON 내보내기)과 복원이 가능하고, 앱 내 ‘전체 초기화’로 모든 데이터를 한 번에 지울 수 있으며, 앱을 삭제하면 기기 내 데이터도 모두 사라집니다. 클린쿨타임은 무료이며 인앱결제가 없고, 특별히 아동을 대상으로 하지 않는 일반 가정용 앱입니다.

주요 기능

🔁

교체 주기 관리

칫솔·수건·정수기 필터 등 생활용품의 교체 주기와 마지막 교체일을 등록해 관리합니다.

🔔

교체일 알림

교체 시기가 다가오면 기기 알림(로컬 알림)으로 미리 안내합니다. (알림 권한 사용)

👪

가족별 관리

가족 구성원 별칭(닉네임)을 만들어 구성원별로 항목을 나눠 관리합니다.

🛒

장보기 + 쿠팡 링크

교체가 필요한 품목의 쿠팡 구매 링크 제공(쿠팡 파트너스 제휴, 사용자가 직접 눌러 이동).

🧾

교체 이력·가격·재고

교체 이력, 가격(선택 입력), 여분 재고 수량을 함께 기록합니다.

📱

기기 내 저장

이용자 데이터는 자체 서버 없이 오직 사용자 기기(IndexedDB)에만 저장. 회원가입·추적 없음.

💾

백업과 복원

JSON 파일로 데이터를 직접 내보내고 복원할 수 있습니다.

🗑️

전체 초기화

앱 내에서 모든 데이터를 한 번에 삭제할 수 있습니다.

제휴 고지. 장보기 화면의 쿠팡 구매 링크는 쿠팡 파트너스 활동의 일환으로, 이를 통해 일정액의 수수료를 제공받을 수 있습니다. 링크는 사용자가 직접 눌렀을 때만 작동하며, 누르면 쿠팡으로 이동합니다. 이때 제휴 추적을 위한 정보가 링크에 포함될 수 있고, 일반 웹 접속과 동일하게 접속 정보(IP 등)가 쿠팡에 전달될 수 있습니다. 앱이 이용자의 개인정보를 쿠팡 측에 능동적으로 전송하지는 않으며, 이동 후 쿠팡 사이트의 개인정보 처리는 쿠팡의 정책을 따릅니다.

개인정보처리방침 요약

시행일·최종 수정일: 2026년 6월 12일. 핵심만 먼저 요약합니다. 전체 조항은 별도 페이지에서 확인하실 수 있습니다.

  • 이용자가 입력한 정보는 오직 이용자 기기 안의 로컬 저장소(IndexedDB)에만 저장되며, 외부 서버로 전송·수집하지 않습니다.
  • 회원가입·로그인이 없고, 개발자나 제3자에게 전달되는 입력 정보가 없습니다.
  • Google Analytics 등 분석 도구, 광고 식별자·광고 SDK·추적 기능을 사용하지 않습니다.
  • 알림은 모두 기기 안에서 로컬로 예약·발송되며 서버 푸시를 쓰지 않습니다. 글꼴·라이브러리가 앱에 모두 포함되어 있어 앱이 자동으로 외부 서버에 접속하지 않으며, 외부 통신은 이용자가 ‘장보기’의 쿠팡 링크를 직접 누를 때에만 일어납니다.
  • ‘장보기’의 쿠팡 링크는 사용자가 직접 누를 때 쿠팡으로 이동하는 제휴 추적 링크이며(쿠팡 파트너스 제휴), 앱이 개인정보를 쿠팡에 능동적으로 전송하지 않습니다.
  • ‘전체 초기화’ 또는 앱 삭제로 데이터를 영구 삭제할 수 있고, JSON으로 백업·복원할 수 있습니다.
  • 특정 연령층, 특히 아동을 대상으로 하지 않는 일반 가정용 앱입니다.
