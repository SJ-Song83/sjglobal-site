1. 총칙: 본 앱은 어떤 앱인가요

클린쿨타임은 가족이 칫솔, 수건, 정수기 필터처럼 ‘교체 주기’가 있는 생활용품을 등록해 두면, 교체 시기가 다가올 때 기기 알림으로 알려주는 가정용 유틸리티 앱입니다.

본 앱은 별도의 회원가입이나 로그인이 없으며, 이용자 데이터를 저장·수집하는 자체 서버를 두지 않습니다. 이용자가 입력한 정보는 모두 이용자의 기기(앱이 설치된 스마트폰 등) 안에만 보관됩니다.

이 방침은 본 앱이 어떤 정보를 다루는지, 그 정보가 어디에 저장되는지, 어떻게 삭제할 수 있는지를 이용자가 알 수 있도록 설명합니다.

2. 수집(저장)하는 정보와 저장 위치

본 앱은 이용자가 직접 입력한 다음 정보를 이용자의 기기 안에 저장합니다.

  • 가족 구성원 별칭(닉네임)
  • 등록한 항목의 이름, 카테고리, 교체 주기
  • 마지막 교체일, 교체 이력
  • 가격 (선택 입력)
  • 여분 재고 수량

저장 위치

위 정보는 모두 이용자 기기 내부의 로컬 저장소(IndexedDB)에만 저장됩니다. 데이터는 기기 안에만 보관되며, 별도의 외부 서버나 클라우드에 저장되지 않습니다. 현재 버전에서는 클라우드 동기화 기능도 제공하지 않습니다.

중요. 위 정보는 이용자의 기기를 벗어나지 않습니다. 즉 개발자(에스제이글로벌)나 제3자가 이 정보를 보거나 가져갈 수 없습니다.

3. 정보의 이용 목적

이용자가 기기에 저장한 정보는 오직 본 앱의 기능을 제공하기 위해서만 기기 안에서 사용됩니다.

  • 등록한 생활용품의 교체 시기를 계산해 알려 주기 위해
  • 가족 구성원별로 담당 항목을 구분해 보여 주기 위해
  • 교체 이력, 여분 재고, 가격 등을 화면에 표시하기 위해

이 정보는 광고, 마케팅, 이용자 분석 등 다른 목적으로 사용되지 않으며, 기기 밖으로 전송되지 않습니다.

4. 이용자 입력 정보의 외부 전송·수집 없음

이용자가 본 앱에 직접 입력한 개인정보는 외부로 전송되거나 수집되지 않습니다. 구체적으로 다음과 같습니다.

  • 개발자(에스제이글로벌)에게 전송되어 수집되는 입력 정보가 없습니다.
  • 제3자에게 전송되어 수집되는 입력 정보가 없습니다.
  • Google Analytics 등 분석 도구를 사용하지 않습니다.
  • 현재 광고 식별자(AAID 등)를 수집하거나 사용하지 않으며, 광고 SDK나 추적 기능이 없습니다.

본 앱은 화면 표시에 사용하는 웹 글꼴과 데이터 저장 라이브러리를 모두 앱 안에 포함(번들)하고 있어, 앱이 자동으로 외부 서버나 콘텐츠 전송 네트워크(CDN)에 접속하지 않습니다. 외부 통신이 일어나는 경우는 이용자가 ‘장보기’ 화면에서 쿠팡 구매 링크를 직접 누를 때뿐이며, 이는 아래 ‘7. 제휴 링크 안내’에서 설명합니다.

참고로 Google Play의 기준에서 ‘수집’은 데이터가 이용자의 기기 밖으로 전송되어 보관·이용되는 것을 뜻합니다. 본 앱은 이용자가 입력한 데이터를 기기 안에서만 처리하고 기기 밖으로 보내지 않으므로, 이 기준상 이용자 데이터를 ‘수집하지 않는’ 앱에 해당합니다.

5. 제3자 제공 및 공유

본 앱은 이용자의 개인정보를 제3자에게 제공하거나 공유하지 않습니다.

본 앱에는 이용자 데이터를 보관하는 자체 서버, 이용자 계정, 광고 SDK, 분석 SDK가 없으므로 제3자와 공유할 입력 개인정보 자체가 존재하지 않습니다.

다만, 본 앱의 ‘장보기’ 화면에서 쿠팡 구매 링크를 누르면 외부 사이트(쿠팡)로 이동하게 됩니다. 이는 이용자가 직접 링크를 눌러 외부 사이트로 이동하는 것일 뿐이며, 본 앱이 이용자의 개인정보를 쿠팡 측에 능동적으로 전송하는 것이 아닙니다. 자세한 내용은 아래 ‘7. 제휴 링크 안내’와 ‘9. 외부 사이트 이동 안내’를 참고해 주세요.

6. 권한 안내 (알림 · 인터넷)

본 앱은 다음 권한을 사용합니다. 각 권한의 사용 목적은 아래와 같습니다.

(1) 알림 권한

교체일이 다가올 때 기기에서 알림으로 안내하기 위해 사용합니다. 모든 알림은 이용자의 기기 안에서 로컬로 예약되고 발송되며, 별도의 서버 푸시를 사용하지 않습니다. 알림과 관련된 어떤 입력 데이터도 외부로 전송되지 않습니다.

(2) 인터넷 접근

  • 이용자가 ‘장보기’의 쿠팡 링크 등 외부 링크를 눌렀을 때 해당 외부 사이트로 이동시키기 위해 사용합니다.
  • 화면에 쓰이는 웹 글꼴과 데이터 저장 라이브러리(idb-keyval)는 모두 앱 안에 포함(번들)되어 있어, 앱이 자동으로 외부에서 내려받지 않습니다.

즉 앱이 자동으로 접속하는 외부 서버는 없으며, 인터넷 접근은 이용자가 직접 외부 링크(쿠팡 등)를 누를 때에만 사용됩니다. 이를 통해 이용자가 입력한 데이터가 수집·전송되지는 않습니다.

7. 제휴 링크 안내 (쿠팡 파트너스)

본 앱의 ‘장보기’ 화면에서는 교체가 필요한 품목의 쿠팡 구매 링크를 제공합니다.

본 앱은 쿠팡 파트너스 활동의 일환으로, 링크를 통해 이동·구매가 발생하는 경우 이에 따른 일정액의 수수료를 제공받을 수 있습니다.

이 링크는 이용자가 직접 눌렀을 때에만 작동하며, 누르면 쿠팡 웹사이트 또는 쿠팡 앱으로 이동합니다. 이때 제휴 추적을 위한 정보가 링크(파라미터)에 포함될 수 있고, 이동 과정에서 일반 웹 접속과 동일하게 접속 정보(IP 주소 등)가 쿠팡에 전달될 수 있습니다. 본 앱이 이용자의 개인정보를 쿠팡 측에 능동적으로 전송하는 것은 아니며(SDK가 아닌 단순 링크 이동입니다), 이동 후 쿠팡 사이트에서 이루어지는 개인정보 처리는 쿠팡의 개인정보처리방침을 따릅니다.

8. 데이터 보관 · 삭제 · 백업

본 앱의 데이터는 이용자의 기기 안에만 저장되므로, 이용자가 직접 보관 여부와 삭제 시점을 결정할 수 있습니다.

보관 기간

이용자가 직접 삭제하기 전까지 기기 안에 보관됩니다. 외부 서버에 별도로 보관되는 사본은 없습니다.

삭제 방법

  • 앱 내 ‘전체 초기화’ 기능을 사용하면 저장된 모든 데이터가 삭제됩니다.
  • 앱을 삭제(언인스톨)하면, 기기 안에 저장되어 있던 데이터도 모두 함께 사라집니다.

이처럼 본 앱의 데이터는 기기 안에만 있으므로, 앱을 삭제하거나 앱 내에서 초기화하면 해당 데이터는 기기에서 영구적으로 삭제됩니다.

백업과 복원

이용자는 앱에서 데이터를 JSON 파일로 직접 내보내(백업) 보관할 수 있고, 필요할 때 이를 불러와(복원) 사용할 수 있습니다. 백업 파일은 이용자가 직접 관리하며, 개발자에게 전송되지 않습니다.

9. 외부 사이트 이동 안내

본 앱에서 쿠팡 구매 링크 등 외부 링크를 누르면, 본 앱을 벗어나 외부 사이트(예: 쿠팡)로 이동하게 됩니다.

외부 사이트에서 이루어지는 개인정보 처리는 본 방침이 아니라 해당 사이트(예: 쿠팡)의 개인정보처리방침이 적용됩니다. 외부 사이트의 개인정보 처리에 대해서는 본 앱과 개발자가 책임지지 않으므로, 해당 사이트 이용 시 그 사이트의 정책을 확인해 주시기 바랍니다.

10. 아동의 개인정보

본 앱은 특정 연령층, 특히 아동을 대상으로 하지 않는 일반 가정용 앱(전체이용가 성격)입니다. 아동 보호를 위한 동의 연령 기준은 국가·관할에 따라 다릅니다(예: 한국 만 14세 미만, 미국 COPPA 만 13세 미만, EU GDPR 만 13~16세 등).

본 앱은 별도의 회원가입이나 외부 전송 없이 기기 안에서만 동작하며, 아동을 포함한 누구의 개인정보도 외부로 수집하지 않습니다.

12. 이용자의 권리

본 앱은 이용자의 개인정보를 외부로 수집·보관하지 않으므로, 이용자는 자신의 데이터를 기기에서 직접 열람하고, 수정하고, 삭제할 수 있습니다.

  • 열람·수정: 앱 화면에서 직접 확인하고 수정할 수 있습니다.
  • 삭제: 앱 내 ‘전체 초기화’ 또는 앱 삭제로 데이터를 지울 수 있습니다.
  • 내보내기: JSON 백업 파일로 직접 내보낼 수 있습니다.

해외에서 본 앱을 이용하는 이용자가 GDPR(유럽), CCPA(미국 캘리포니아) 등에 따른 열람·삭제 등의 권리를 행사하려는 경우, 본 앱은 개인정보를 외부에 보관하지 않으므로 위와 같이 이용자가 기기에서 직접 권리를 행사할 수 있습니다. 추가 문의가 있으면 위 문의처로 연락해 주시기 바랍니다.

13. 개인정보의 안전성 확보조치

본 앱은 이용자의 개인정보가 기기를 벗어나지 않도록 설계하는 것 자체를 핵심적인 안전성 확보조치로 삼고 있습니다. 구체적인 조치는 다음과 같습니다.

  • 이용자 데이터를 보관하는 자체 서버, 이용자 계정, 외부 전송 채널을 두지 않습니다. 따라서 전송 과정에서 정보가 유출될 위험이 구조적으로 발생하지 않습니다.
  • 이용자가 입력한 데이터는 모바일 운영체제가 앱별로 분리(샌드박스)해 보호하는 기기 내부의 로컬 저장소(IndexedDB)에만 저장됩니다.
  • 화면 표시에 쓰이는 웹 글꼴과 데이터 저장 라이브러리를 모두 앱 안에 포함(번들)하여, 앱이 자동으로 외부 서버에 접속하지 않도록 했습니다.
  • 이용자가 직접 내보내는 백업 파일(JSON)은 이용자가 보관·관리하며, 개발자에게 전송되지 않습니다.

본 앱은 외부에 보관하는 개인정보가 없으므로, 서버 접근권한 관리나 전송 구간 암호화 같은 서버 기반 보호조치는 본 앱에 해당되지 않습니다. 대신 위와 같이 데이터를 기기 안에서만 처리하는 구조로 안전성을 확보합니다.

14. 고지 의무 (시행일 및 변경 시 공지)

본 개인정보처리방침의 시행일은 2026년 6월 12일이며, 같은 날을 최종 수정일로 합니다.

법령이나 앱 기능의 변경 등으로 본 방침의 내용이 바뀌는 경우, 변경 사항과 개정일을 본 방침이 게시된 공개 페이지(https://sjglobal.inclicks.kr/privacy.html)에 게시하여 안내하며, 게시 시점부터 효력이 발생합니다. 중요한 변경이 있을 때에는 이용자가 알아보기 쉽도록 변경 내용을 함께 게시합니다.

현재 본 앱에는 광고가 없습니다. 향후 광고가 도입되는 경우, 광고 제공에 필요한 정보(예: 광고 식별자) 처리 내용을 반영하여 본 방침을 개정하고, 개정 사항을 본 페이지에 사전 고지하며, 같은 시점에 Google Play 데이터 보안(Data safety) 정보도 실제 처리 내용에 맞게 갱신합니다.
